CSS
( Cascading Style Sheet ) Significa folha de estilo em cascata , isso porque a estilização segue da menor importância para maior importância , deixando o HTML mais bonito , ele é basicamente a pele de todo o site. CSS tem uma sintaxe simples, e utiliza uma série de palavras em inglês para especificar os nomes de diferentes propriedade de estilo de uma página. Uma instrução CSS consiste em um seletor e um bloco de declaração. Cada declaração contém uma propriedade e um valor, separados por dois pontos (:). Cada declaração é separada por ponto e vírgula (;)

HTML
(Hyper Text Markup Language) Linguagem de marcação de texto; O HTML foi criado por Tim Berners-Lee e Robert Cailliau , a ideia era criar um Texto ao qual se agregariam outras informações como os links , facilitando a troca , o agrupamento e as ,atualizações de informações de documentos. O HTML é considerado o corpo de um documento por isso é a base para o Desenvolvimento Web sem o básico do HTML não conseguimos fazer nada ; É correto escrever o HTML seguindo a semântica , que é a forma de deixar o site com suas informações bem explicadas e compreensíveis para o computador,

JAVASCRIPT
Como surgiu o Javascript ? O JS foi criado em 1995 por " Brendan Eich " que na época trabalhava no Netscape ; A linguagem foi criada em apenas 10 dias , A função do JS era fazer coisas básicas como validar formulários , acessar o DOM , que na época era algo realmente novo e incrivel para os Desenvolvedores , O JS teve outros nomes antes de ser JavaScript , Macha foi o primeiro e depois LiveScript ; Com o sucesso da linguagem de programação Java os criadores resolveram colocar Javasript pegando uma carona na popularidade do Java ; Em 1996 surgiu a versão padronizada do JS chamada ECMAscript O Javascript nasceu como uma linguagem para rodar nos navegadores do lado dos clientes , mais hoje ele roda também no servidor .
